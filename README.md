# Nvim Config
Min Nvim konfigurasjon.


## Første gang bruk
- **MacOS:** `brew install nvim`
- **Windows:** `winget install Neovim.Neovim`

Feil melding vises når nvim åpnes. Da installerer jeg plugins med "Plug":
```sh
:PlugInstall
```

Etter på, installerer jeg COQ dependencies:
```sh
:COQdeps
```
